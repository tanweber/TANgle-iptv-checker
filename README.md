# TANgle - IPTV Checker

Веб-интерфейс для умного управления IPTV-плейлистами.
TANgle автоматически собирает единый M3U-плейлист из нескольких источников, проверяет каналы на доступность и скорость отклика, удаляет дубли и формирует актуальную ТВ-программу (EPG) только для работающих каналов.

Этот проект был разработан с использованием MiMo Code —新一代 AI-помощника для программистов от Xiaomi.

## Возможности

- **Проверка IPTV-каналов** — параллельное тестирование доступности каналов из M3U и M3U8-плейлистов с измерением времени отклика.
- **Формирование единого плейлиста** — объединение нескольких плейлистов в один с возможностью гибкой настройки правил фильтрации по скорости и доступности.
- **Автоматическое удаление дублей** — перед публикацией итогового плейлиста все повторяющиеся каналы исключаются.
- **ТВ-программа (EPG)** — автоматическое скачивание и слияние XMLTV-файлов из разных источников с формированием программы только для каналов, присутствующих в итоговом плейлисте.
- **Статистика и мониторинг** — отслеживание доступности каналов, истории подключений и динамики изменения качества источников.
- **Адаптивный веб-интерфейс** — удобное управление с ПК, планшетов и смартфонов.
- **7 тем оформления** — Dark, Light, Monokai, Dracula, Nord, Solarized, GitHub на ваш выбор.
- **Автоматизация** — встроенный планировщик (шедулер) для регулярной проверки плейлистов и обновления EPG.
- **Простота развертывания** — готовый Docker-образ для запуска одной командой.

## Быстрый старт

```bash
git clone https://github.com/ваш-аккаунт/tangle.git
cd tangle
docker compose up -d
```

Откройте в браузере: `http://localhost:9239`

Логин по умолчанию: `admin` / `admin`

## Структура проекта

```
tangle/
├── app.py              # FastAPI сервер
├── database.py         # SQLite операции
├── checker_core.py     # Проверка каналов
├── epg.py              # Обработка EPG
├── static/
│   ├── index.html      # Основной интерфейс
│   └── login.html      # Страница входа
├── Dockerfile
├── docker-compose.yml
└── nginx.conf
```

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Веб-интерфейс |
| GET | `/rus_fixed.m3u` | Скачать плейлист |
| GET | `/epg.xml` | Скачать ТВ-программу |
| POST | `/api/login` | Авторизация |
| GET | `/api/stats` | Статистика каналов |
| GET | `/api/channels` | Список каналов |
| GET | `/api/sources` | Источники плейлистов |
| GET | `/api/epg-sources` | Источники EPG |
| POST | `/api/check` | Запустить проверку |
| GET | `/api/access/log` | Лог подключений |

## Настройки

- Интервал проверки каналов
- Интервал обновления EPG
- Правила формирования плейлиста (по скорости и доступности)
- Период расчёта доступности каналов
- Возможность добавить плейлисты
- Возможность добавить ТВ-программа (EPG)

## Требования

- Docker и Docker Compose
- 512MB RAM
- 1GB дискового пространства
