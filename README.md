# TANgle - IPTV Checker

Веб-интерфейс для проверки IPTV-плейлистов с автоматическим формированием ТВ-программы.

## Возможности

- **Проверка каналов** — параллельная проверка доступности каналов из M3U/M3U8 плейлистов
- **ТВ-программа (EPG)** — автоматическое скачивание и мерж XMLTV-программ из нескольких источников
- **Статистика** — отслеживание доступности каналов, времени отклика, истории подключений
- **7 тем** — Dark, Light, Monokai, Dracula, Nord, Solarized, GitHub
- **Адаптивный интерфейс** — корректно работает на мобильных устройствах
- **Автоматизация** — шедулер проверок и обновления EPG
- **Docker** — простой запуск одной командой

## Быстрый старт

```bash
git clone https://github.com/ваш-аккаунт/tangle.git
cd tangle
docker compose up -d
```

Откройте в браузере: `http://localhost:9239`

Логин по умолчанию: `admin` / `admin`

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Веб-интерфейс |
| GET | `/rus_fixed.m3u` | Скачать плейлист |
| GET | `/epg.xml` | Скачать ТВ-программу |
| POST | `/api/login` | Авторизация |
| GET | `/api/stats` | Статистика каналов |
| GET | `/api/channels` | Список каналов |
| GET | `/api/sources` | Источники плейлистов |
| GET | `/api/epg-sources` | Источники EPG |
| POST | `/api/check` | Запустить проверку |
| GET | `/api/access/log` | Лог подключений |

## Настройки

- Интервал проверки каналов
- Интервал обновления EPG
- Правила формирования плейлиста (по скорости и доступности)
- Период расчёта доступности каналов

## Требования

- Docker и Docker Compose
- 512MB RAM
- 1GB дискового пространства
